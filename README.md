# **KenSaku**
A web application that allows users to create a simple and customizable search page.

You are now in the **BACKEND REPOSITORY**. Explore more at:
* [Frontend repository](https://github.com/yozakura-minato/kensaku-fe)
* [Document repository](https://github.com/yozakura-minato/kensaku-docs)

### **Technology Stack**
* Spring Boot
* Java 25
* Gradle
* PostgreSQL

### **Set Up**
1. Create a local PostgreSQL database.
2. Follow instructions in ```application-example.yaml``` file.
